# Next.js Practice Project

Project ini merupakan latihan penggunaan Next.js dan Git untuk memahami dasar pembuatan aplikasi web modern.

## 1. Install Dependencies

Pastikan Node.js sudah terinstall di komputer.

Kemudian jalankan perintah berikut di terminal:

npm install

Perintah tersebut akan menginstall semua dependencies yang terdapat pada file package.json.

## 2. Menjalankan Development Server

Untuk menjalankan project dalam mode development gunakan perintah:

npm run dev

Setelah server berjalan, buka browser dan akses:

http://localhost:3000

## 3. Struktur Folder Project

Berikut struktur folder utama pada project:

- app/ → berisi halaman dan routing aplikasi Next.js  
- public/ → berisi file statis seperti gambar dan icon  
- node_modules/ → berisi semua dependencies project  
- package.json → konfigurasi project dan daftar dependencies  
- README.md → dokumentasi project
